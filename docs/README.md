# Chhaya Platform

## Overview
Chhaya is a multi-modal creative intelligence platform for text, image, and audio. It adapts over time through a persistent creative memory and feedback loops.

## Structure
- `frontend/`: React app
- `backend/`: Node.js API
- `docs/`: Documentation

## Quick Start
- Run backend: `npm install` then `npm run dev`
- Run frontend: `npm install` then `npm run dev`
