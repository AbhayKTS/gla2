## Design System
- Colors: Deep midnight (#0b0c13), Electric violet (#8c6bff), Cyan (#44d0ff), Amber (#ffd370)
- Typography: Inter / Segoe UI
- Components: cards, sliders, tags, panels, split workspaces

- Feedback: rate, edit, re-run with constraints.
- Memory: lock/unlock preferences to guide future generations.
